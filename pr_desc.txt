- `testSanitizeFilename_onlyDisallowedCharacters_replacesAllWithUnderscore`: Validates edge cases with filenames composed entirely of disallowed characters.
- `testSanitizeFilename_emptyString_returnsEmptyString`: Confirms the function handles empty strings gracefully without crashing.

✨ **Result:**
The test coverage allows for confident refactoring of `DocumentFileUtilities` without introducing regression. The tests provide a safety net for any changes related to handling user-provided file names.
